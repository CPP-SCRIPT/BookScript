# VirtualBox and WSL setup

## VirtualBox Setup

Download VirtualBox [here](https://www.virtualbox.org/wiki/Downloads)

For Linux Security Purposes you can download UbuntuSEEDLABS here [https://seedsecuritylabs.org/lab\_env.html](https://seedsecuritylabs.org/lab_env.html)

For instructions [https://seedsecuritylabs.org/Labs\_16.04/Documents/SEEDVM\_VirtualBoxManual.pdf](https://seedsecuritylabs.org/Labs_16.04/Documents/SEEDVM_VirtualBoxManual.pdf)

## Windows Subsystem Linux Setup

First go to Windows Features and check mark the windows subsystem Linux box.

Then go to the microsoft store and download the appropriate linux distribution then click on the distribution and then start. 

[https://www.microsoft.com/en-sg/store/apps](https://www.microsoft.com/en-sg/store/apps)
