# Linux Commands Guide

#### cd :: Change Directory

```text
cd . :: Move to next directory
cd .. :: Move to previous directory
cd ~
cd - 
cd /
```

#### pwd :: print working directory

**ls :: list all files**

**clear :: clear screen**

**cat :: prints** 

**echo :: prints to screen**

**mv :: move** 

**cp :: copy**

**nano, vim ,vi ,pico :: text editors**

**gcc :: c compiler**
